== Description ==

Allows the use of special short codes [padlet] and [wallwisher] for embedding Padlet walls. The short code can be found in the _Share_ section of a wall's sidebar on padlet.com

**Example:** 
`[padlet key="8ftvxj" height="480" width="100%"]`

or 

`[wallwisher key="8ftvxj" height="480" width="100%"]`


== Installation ==

__For old school manual installation__: copy the folder "padlet-shortcode" into the `/wp-content/plugins/` folder. Then go to the `Plugins` area of the Admin and activate. 

__For automatic installation__: Goto `Plugins > Add` New in the admin area of your wordpress blog. Search for _"Padlet Shortcode"_ and install the plugin that shows up in the search results.
